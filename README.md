# C-Game

## EN

This game is a simple C++ application and you can move the star (*) on the console screen with arrow keys.

I developed it using object-oriented programming principles to learn it.

#### Features:

Coordinate management using a Point class

Multiple constructors

Move the star on console screen

Arrow key input detection


## TR

C++ ve OOP'de yeni öğrendiğim bilgileri kullanarak bir yıldız hareket ettirme oyunu yaptım. Ok işaretlerine basarak yıldızı konsol üzerinde hareket ettirebiliyoruz. Bunu yapma amacım hem OOP programlama prensiplerini öğrenmek hem de bilgileri tekrar etmek.

#### Özellikler:

Koordinatları Point classı ile yönetmek

Birden fazla constructor kullanmak (neredeyse bütün constructorlar kullanılıyor)

Yıldızı konsolda hareket ettirmek

Ok işaretine basıldığı zaman getch ile algılamak
